# 我的 GitHub Pages 网站

这是一个托管在 GitHub Pages 上的静态网站。

## 🌐 访问地址

网站部署后可通过以下地址访问：
- `https://<你的用户名>.github.io/<仓库名>/`

## 📁 项目结构

```
├── index.html    # 网站首页
└── README.md     # 项目说明文档
```

## 🚀 如何部署

1. 进入仓库的 **Settings** 页面
2. 在左侧菜单找到 **Pages**
3. 在 **Source** 下选择 `main` 分支
4. 点击 **Save**
5. 等待几分钟后，网站即可访问

## 🛠️ 本地预览

直接在浏览器中打开 `index.html` 文件即可预览。
