# 更新升级指南

## 脚本更新

运行安装脚本，选择 `2` 进行更新：

```bash
curl -fsSL https://6qu.cc/dns2/install.sh | bash
```

```
========================================
       六趣DNS 管理脚本                 
========================================

  1. 安装
  2. 更新    ← 选择此项
  3. 卸载
  0. 退出

========================================
```

选择 `2` 后，系统会自动拉取最新镜像并重启容器，配置文件不会丢失。

---

## 手动更新

如果脚本更新失败，可以手动更新：

```bash
cd /opt/dns && docker-compose pull && docker-compose down && docker-compose up -d
```

---

## 常见更新问题

### 1. 更新失败

- 检查网络连接是否正常
- 查看日志了解具体错误：`docker logs dns-app`

### 2. 更新后页面异常

- 清除浏览器缓存
- 或强制刷新：Ctrl + Shift + R
