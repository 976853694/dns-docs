# DNS 概述

DNS（Domain Name System）是互联网的"电话簿"，将域名转换为 IP 地址。

## 什么是 DNS

DNS 是一个分布式数据库系统，主要功能包括：

- **域名解析**：将域名转换为 IP 地址
- **反向解析**：将 IP 地址转换为域名
- **邮件路由**：通过 MX 记录指定邮件服务器

## DNS 工作原理

```
用户 → 本地 DNS → 根服务器 → TLD 服务器 → 权威服务器 → IP 地址
```

### 解析流程

1. **浏览器缓存**：首先检查浏览器缓存
2. **系统缓存**：检查操作系统 DNS 缓存
3. **本地 DNS**：查询本地 DNS 服务器
4. **递归查询**：本地 DNS 进行递归查询
5. **返回结果**：将 IP 地址返回给用户

## DNS 层级结构

```
.                    (根域)
├── com              (顶级域 TLD)
│   ├── example      (二级域)
│   │   └── www      (子域)
│   └── google
├── org
└── cn
```

## 常用 DNS 服务器

| 服务商 | IPv4 | IPv6 |
|--------|------|------|
| Google | 8.8.8.8 | 2001:4860:4860::8888 |
| Cloudflare | 1.1.1.1 | 2606:4700:4700::1111 |
| 阿里云 | 223.5.5.5 | 2400:3200::1 |

## 相关概念

- **TTL**：Time To Live，缓存时间
- **SOA**：Start of Authority，权威记录
- **NS**：Name Server，域名服务器
